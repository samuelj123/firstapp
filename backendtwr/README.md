# Backend for TWR App

## Description

Definitely made using [Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Setting up a Development Environment

### Prerequisites
  * Nodejs
  * npm

```bash
$ npm install 
# It installs all the node_modules

$ npm run start:dev
```

## Running it on a Server 


```bash
$ npm install
# It installs all the node_modules 

$ npm run build

$ npm run start:prod
```
